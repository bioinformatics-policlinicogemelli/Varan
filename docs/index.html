VARAN

Overview

Varan is a Python-based application that provides a pipeline to automatically prepare and manipulate cancer genomics data in the specific format supported by the cBioPortal.

Features

  • Study Creation: This section provides, starting from raw vcf files, a well-structured and validated study folder ready to be uploaded into the local instance of cBioPortal.

  • Study Manipulation: This section allows users to work on already existing studies. In particular it is possible to merge two studies or to modify a study by extracting/removing samples.

Varan quick start

Quick deploy Varan from raw data to launching analysis pipelines in under 30 minutes using a Docker container.

User guide

User guide provides in depth description of Varan usage, detailing specific functions, workflows and use cases.
