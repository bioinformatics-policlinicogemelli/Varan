VARAN

Overview

Varan is a Python-based application that provides a pipeline to prepare and manipulate cancer genomics data for cBioPortal.

Features

  • Study Creation: Generates a validated study folder from raw VCF files, ready for upload to cBioPortal.
  • Study Manipulation: Allows merging studies or modifying an existing study by extracting or removing samples.

Quick Start

Deploy Varan from raw data to analysis pipelines in under 30 minutes using a Docker container.
